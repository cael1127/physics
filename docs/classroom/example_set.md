# Classroom Example Set

Reference examples for labs and lectures:

- `examples/classroom/example_queries.c`
  - Demonstrates point and AABB query APIs.
- `examples/classroom/example_replay.c`
  - Demonstrates snapshot save/load replay workflow.
- `tests/test_constraints.c`
  - Demonstrates distance joint behavior for constraint lessons.
- `tools/fullphysics_headless.c`
  - Demonstrates deterministic headless experiment runs with CSV outputs.

These examples are intentionally small so students can modify and rerun them quickly.
