# FullPhysicsC Docs

This documentation is organized for research and education workflows.

- **Learn** (`docs/learn/`): progressive physics engine concepts and hands-on labs.
- **Use** (`docs/use/`): practical integration and reproducible experiment execution.
- **Contribute** (`docs/contribute/`): architecture and extension guides for contributors.
- **Classroom** (`docs/classroom/`): ready-to-run teaching modules and assignments.
- **Pinned baselines** (`docs/reproducibility/baselines.md`): optional golden metric CSVs for strict regression checks.

Recommended order for new users:

1. `docs/learn/01_integration.md`
2. `docs/learn/02_collision_and_contacts.md`
3. `docs/learn/03_constraints.md`
4. `docs/learn/04_multiphysics.md`
5. `docs/learn/05_reproducibility.md`
